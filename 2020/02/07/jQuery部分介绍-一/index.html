jQuery部分介绍(一)

什么是jQuery

- 一个jQuery的框架
- 兼容各种主流的浏览器

jQuery的好处

  • 简化了js的操作
  • 不需要担心兼容的问题
  • 提供了大量使用的方法

    如何学习jQuery

  1. jQuery的官方网站

  2. jQuery只是辅助工具，依旧要学会原生js

  3. 分阶段学习

    jQuery版本的区别：

    1.0：可以兼容ie8以下的浏览器
2.0：只兼容ie8以上的
3.0

    函数太多，遇到新函数查阅中文文档

    jQuery的设计思想 -选择网页元素

  4. 模拟css选择网页元素

    • $(“#div1”).css(“backgroundColor”,”red”);//其他选择器也可以使用

  5. 独有的表达式

    • $(“li:first”).css(“backgroundColor”,”green”)
    • $(“li:last”).css(“backgroundColor”,”green”)
    • $(“li:even”).css(“backgroundColor”,”green”)//下标为偶数的li标签都变色
    • $(“li:odd”).css(“backgroundColor”,”green”)//下标为奇数的li标签都变色
    • $(“li:eq(2)”).css(“backgroundColor”,”green”)//下标为2的元素变色
    • 还有很多类似的用法

  6. 多种筛选方法

    • $(“li:eq(2)”).css(“backgroundColor”,”green”)//下标为2的元素变色
    • $(“li”).eq(2).css(“backgroundColor”,”green”)//和上面的功能一样
    • $(“li.box”).css(“backgroundColor”,”green”)//li标签内。box的元素变色
    • $(“li”).filter(“.box”).css(“backgroundColor”,”green”)//功能与上者相同

      jQuery的设计思想 -jQuery的写法

      在jQuery中基本见不到等号。所有的赋值几乎都是通过函数传参的方式来实现的

  7. 方法函数化

    • window.onload在jQuery中可以直接用$(function(){xxxx});来代替//xxxx里面就是你要写的函数功能
    • onclick事件在jQuery中可以通过$(“h1”).click(function(){xxxx});来代替//后续事件绑定都是一样的套路
    • $(“h1”).mouseover(function(){
      this.style.backgroundColor='red';
      })
      $(“h1”).mouseout(function(){
      this.style.backgroundColor='green';
      })//用来诠释上面的例子

  8. 链式操作

    • $(function(){
      $(“h1”).click(function(){
      alert(“你要的函数功能”)
      });
      $(“h1”).mouseover(function(){
      this.style.backgroundColor=’red’;
      })
      $(“h1”).mouseout(function(){
      this.style.backgroundColor=’green’;})
      $(“h1”).css(‘backgroundColor’,”blue”)
      })//原来的代码，均对同一个标签操作，那么我们可以优化代码

    • $(function(){
      $(“h1”).click(function(){

      alert("你要的函数功能")

      })
      .mouseover(function(){

      this.style.backgroundColor='red';

      })
      .mouseout(function(){

      this.style.backgroundColor='green';})

      .css(‘backgroundColor’,”blue”)
      })//与上面代码功能相同

    • alert($(“h1”).html())

      我裂开了

      “)
      //以上两行代码函数使用的都是同一个，只是一个传递了参数一个没有

      jQuery设计思想 -jQuery和JavaScript的关系

  10. 可以共存，但是不能混用，因为jq实际上是js的OOP封装，所以在jq对象里只能使用jq的方法

    $("h1").innerHTML;//错误的
$("h1").html();//正确的

    jQuery的常用方法

  11. filter 过滤 对已经获取到的元素进行过滤

    $(“div”).filter(“.box”).css(“backgroundColor”,”blue”);//div中，class为box的

  12. not filter的反义词

    $(“div”).filter(“.box”).css(“backgroundColor”,”blue”);//div中，class不为box的

  13. has 拥有，直接判断子节点是否要符合条件的元素

    $(“div”).has(“.box”).css(“backgroundColor”,”blue”);//div中，有子节点class为box的

  14. prev 查找当前兄弟节点中的上一个节点

    $(“div1”).prev().css(“backgroundColor”,”blue”);//寻找div1的前一个节点

  15. next 查找当前兄弟节点中的下一个节点

    $(“div1”).next().css(“backgroundColor”,”blue”);//寻找div1的后一个节点

  16. find 查找当前节点的子节点

    $(“ul”).find(“li”).css(“backgroundColor”,”blue);//找到ul标签下的li标签

    $(“ul”).find(“[name=hello]”).css(“backgroundColor”,”blue);//找到ul标签下的name=hello的标签

  17. index 获取当前节点在兄弟节点中的下标

    $(“h3”).index()//获取下标

  18. eq 通过下标获取指定的元素节点

    $(“li”).eq(2).css(“backgroundColor”,”blue);

    $(“li:eq(2)”).css(“backgroundColor”,”blue);

  19. attr 设置和修改行间的属性

    alert($(“#div1”).attr(“id”);)//获取#div1的id

    $(“#div1”).attr(“title”,”box”)//修改了title属性

    $("#div1").attr({
    title:'world',
    class:'xxx',
})//可以这样修改多条属性
