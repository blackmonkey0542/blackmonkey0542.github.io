python爬取免费网络小说(难度：极度easy)

python爬虫入门

什么是网络爬虫?-一段程序or脚本

需要那么一丢丢的html基础
可以看之前那篇初识html的博客了解一点

爬虫能干什么？-模拟浏览器自动浏览,自动批量的采集我们需要的资源

使用环境：python 3.0
使用模块： requests(第三方库文件)

+ cmd下 pip install requests 安装

常用请求：get/post
一段爬指定网页的测试代码
采用的方法是向服务器发送get请求，然后规定编码方式，把html爬下来

开发爬虫的步骤

+ 目标数据
    - 页面
    - 网站
+ 分析数据加载流程
    - 分析目标数据所对应的url
+ 下载数据
+ 清洗，处理数据
+ 数据持久化
+ 写入文件

正则表达式

r'你要的内容前的东西(.*?)你要的内容后的东西',(你要读取的内容)

本文以爬取精彩小说网为例子

——————由于对面加了反爬虫，于是我们需要加上headers来冒充浏览器浏览，非否则请求会被驳回ERR 403
header可以在浏览器中F12中找到
