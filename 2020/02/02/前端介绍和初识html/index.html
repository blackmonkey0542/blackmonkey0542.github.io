前端开发介绍和初识html

什么是前端开发?

前端开发是创建Web页面或app等前端界面呈现给用户的过程，通过HTML，CSS及JavaScript以及衍生出来的各种技术、框架、解决方案，来实现互联网产品的用户界面交互。它从网页制作演变而来，名称上有很明显的时代特征。在互联网的演化进程中，网页制作是Web1.0时代的产物，早期网站主要内容都是静态，以图片和文字为主，用户使用网站的行为也以浏览为主。随着互联网技术的发展和HTML5、CSS3的应用，现代网页更加美观，交互效果显著，功能更加强大。

前端的三座大山

1. 超文本标记语言(html)：基本功能区
2. 层叠样式表(css)：用来设置各种样式，居中等属性，可通过id，或者class来找到html里的元素
3. JavaScript编程语言(js)：用来执行狠很多操作，编程语言能够处理一些data来方便交互

html

什么是html？

HTML 是用来描述网页的一种语言。
HTML 指的是超文本标记语言 (Hyper Text Markup Language)
HTML 不是一种编程语言，而是一种标记语言 (markup language)
标记语言是一套标记标签 (markup tag)
HTML 使用标记标签来描述网页

那么什么是html的标签

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
HTML 标签是由尖括号包围的关键词，比如 <html>
HTML 标签通常是成对出现的，比如 <b> 和 </b>
标签对中的第一个标签是开始标签，第二个标签是结束标签
开始和结束标签也被称为开放标签和闭合标签

空的html元素

空的 HTML 元素
没有内容的 HTML 元素被称为空元素。空元素是在开始标签中关闭的。
<br> 就是没有关闭标签的空元素（<br> 标签定义换行）。
在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须被关闭。
在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。
即使 <br> 在所有浏览器中都是有效的，但使用 <br /> 其实是更长远的保障。

杂谈

在编写html的时候，可以经常查阅不同标签的功能来使用，所有的标签的作用大同小异，html只是用来设置网页的基本结构，纯html也会很丑，所以我们会在后来引入css来美化，但是，html是我们构建网页的第一步，也是很重要的一步，功能区怎么划分，直接决定了后期功能实现的可能性。也会影响我们和理想中界面实现的一致性。所以html要熟知如何使用，并且学会不同的容器来装不同的内容，为后期维护等做好基础。
