MySQL的笔记(一)

MySQL

大多数据库基于标准的SQL标准

select 语句

  1. AS语句的用法：

    select 'XXX' as xxx,
       'YYY' as yyy;

    CONCAT 连接字符串 :CONCAT(`A`, `B`) AS C;concat会在遇到NULL的时候就把整个字符串变为NULL
    即 NULL + ‘string’ = NULL,所以利用IFNULL进行一个转义的操作. CONCAT(
    `A`,
    ‘.’,`B`,
    ‘.’,IFNULL(`C`,0),
    ‘.’,IFNULL(`D`,0),
    ‘.’,IFNULL(`E`,0),
    ‘.’,IFNULL(`F`,0));

distint去重：SELECT DISTINCT `department_id` ;

  1. 条件：

  • 和VB一样啊….就> < = != >= <=

    -where i<>10

  1. 逻辑

    • 这个和vb或者c就不一样了(and or not)
    • while A and B(差不多就是这个用法)

      筛选的另外一些办法

  • 通配符%(任意数量)_(一个字符) ：WHERE `A` LIKE '%b%';

    where A = ‘a’ or A = ‘b’ or A = ‘c’;
    和一样的
    where A in(‘a’,’b’,’c’);

  • 判断是否为空：

    • where `A` IS NOT NULL
    • where `A` IS NULL
    • where `A` <=> NULL

  • 升降序的办法

    • ORDER BY salary ASC;

      • ascend 升序

    • ORDER BY salary DASC;

      • descend 降序

        长度LENGTH（）

        可以用来统计字符串长度啥的，然后对其进行排序，比如：order by length(A) ASC;

阮老板的参考还有好多，想看详细的点点看呗

  • 常用字符串函数 SUBSTR REPLACE INSERT 里面都有

  • TRIM函数：去空格的函数

    TRIM(string) trim 修剪 去掉string左右两边的空格

    LTRIM(string) left trim 修剪 去掉string左边的空格

    RTRIM(string) right trim 修剪 去掉string右边的空格

    TRIM( childStr FROM string) trim 修剪 去掉string左右两边的子串childStr

    TRIM(LEADING childStr FROM string) LEADING 领先的 去掉string前边的子串childStr

    TRIM(TRAILING childStr FROM string) TRAILING 末尾的 去掉string后边的子串childStr

  • PAD函数(用来补字符串)：
    LPAD(str, len, pad) left 左边 用字符串pad对str最左边进行填充，直到str的长度为len个字符

    RPAD(str ,len, pad) right 右边 用字符串pad对str最右边进行填充，直到str的长度为len个字符
